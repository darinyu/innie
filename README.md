# Innie

Innie gives every software system a work self: humans stay the outie, the
Slack bot is the innie, and agent harnesses do the work behind the boundary.

Innie is a harness-neutral sidekick layer for AI agents. It makes runtimes such
as Codex, Claude Code, OpenCode, Goose, or future harnesses usable as controlled
work assistants without turning Innie into another agent loop.

See [docs/initial-plan.md](docs/initial-plan.md) for the initial product and
architecture plan.
