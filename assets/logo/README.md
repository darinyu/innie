# Innie Logo

The current logo direction is **Black Ice**, a flat percent-inspired mark:

- The upper-left dot is the outie.
- The slash is the boundary and task path.
- The lower-right dot is the innie.

## Files

- `innie-mark.svg`: primary mark.
- `innie-app-icon.svg`: square app icon for Slack, GitHub, and favicons.
- `innie-wordmark.svg`: mark plus lowercase wordmark.
- `innie-mark-mono.svg`: monochrome fallback.

## Palette

- Ink: `#05070A`
- Ice: `#EEF6FF`
- Innie blue: `#93C5FD`
