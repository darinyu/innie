# Innie

Innie is a harness-neutral sidekick layer for AI agents.

The project goal is to make agent harnesses such as Codex, Claude Code,
OpenCode, Goose, or future runtimes usable as controlled work assistants
without turning Innie into another agent loop.

See [docs/initial-plan.md](docs/initial-plan.md) for the initial product and
architecture plan.
